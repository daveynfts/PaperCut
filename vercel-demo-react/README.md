# PaperCut frontend

React/Vite interface for the PaperCut API. See the repository root `README.md`
for setup, authentication, payment modes, and verification commands.

```bash
npm install
npm run dev
```

Before production deployment, set `VITE_PRIVY_APP_ID` and `VITE_API_URL` in the
hosting environment. Run `npm run check` to lint and create a production build.
